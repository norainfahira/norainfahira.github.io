    SECR1013 - Digital Logic

    Credit Hours : 3

    Lecturer : Dr. Raja Zahilah binti Raja Mohd Radzi

    Section : 03

    Course Description

    This course introduces the fundamentals of digital electronics and digital logic, covering number systems, logic gates, Boolean algebra, and combinational and sequential circuits, with hands-on experience using simulation tools and real digital systems.

    Lab Activities

    Lab 1 : Combinational Logic Completed

    Objectives

    The objectives of this laboratory are to introduce students to :

    • basic bread boarding and wiring techniques
    • the use of input switches and outputs LEDs in generating truth tables for a combinational logic circuit
    • to verify the characteristic of the basic gates
    See here !
    Lab 2 : Combinational Digital Circuit Design Simulation using Dedds SimulatorCompleted

    Objectives

    To expose student with :

    • producing digital logic circuit, generating truth table and Timing Diagram with Deeds Simulator
    • a complete cycle process of combinational circuit design and simulate with Deeds Simulator
    Lab 3 : Synchronous Digital CounterCompleted

    Objectives

    Student have to :

    • implement a synchronous counter circuit into physical circuit using Breadboard, Flip-Flops, Basic Gates and Switches
    • completing the next-state table of the counter circuit
    • sketch the state diagram of the counter circuit
    • identify the properties of the counter

    Projects

    Electronic Lift/Elevator Controller System (Simulation Using Deeds Software)Completed

    Objectives

    Students have to :

    • design and implement a secure password-protected lift system
    • incorporate combinational circuit components such as comparators, decoders, multiplexers, and basic logic gates for input processing, floor comparison, and output control
    • utilise sequential circuit elements including counters, flip-flops, latches, and logic gates to accurately control floor counting, movement direction, and door operations
    • verify correct operation through simulation and testing in DEEDS
    • showcase the application of digital electronics principles in the context of lift

    SECI1013 - Discrete Structure

    Credit Hours : 3

    Lecturer : Dr. Mohamad Syukor bin Talib

    Section : 03

    Course Description

    This course introduces the fundamental concepts of discrete structures in computer science, including set theory, relations, functions, counting methods, graph theory, trees, and finite automata.

    Assignments

    Assignment 1 : Chapter 1 - Set Theory and Logic Completed
    Assignment 2 : Chapter 2 - Relation, Function and Recurrence Completed
    Assignment 3 : Chapter 3 - Counting and Discrete Probability Theory Completed
    Assignment 4 : Chapter 5 - Finite Automata Completed

    Learning Outcomes :

    • Apply set theory, relations, and functions to solve computer science problems
    • Analyze and solve problems using recurrence relations and counting methods
    • Use graph theory and trees to model and solve real-world problems
    • Apply finite automata and finite state machines to model electronic devices and systems
